## 한동 친구 찾기, 나의 친구가 되어줘!
한동 친구 찾기 프로그램은 유저와 한동대학교 재학생들의 유사도 정보를 바탕으로 친구를 매칭해줍니다.

![young-people-g194636693_1920](https://user-images.githubusercontent.com/67420453/166155520-7be66bfe-4ab8-4b52-91c4-6724d47f5dd0.jpg)

### 프로그램 기능
1. 사용자 검색
2. 유사도 계산
3. 친구 추가 및 관리
4. 메시지 전송 및 확인

### 개발환경 및 언어
개발 환경: VSCode
언어: C
